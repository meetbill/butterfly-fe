编译 Build

使用 grunt编译你的项目，以及使用 bower管理依赖。

环境要求

为了使用 Grunt您需要在您的计算机上安装 Node.js，并确保其是最新版本。 确保您的 NPM已安装。
您可以通过打开终端（Linux 或 Mac）或命令提示符（Windows）来检查这些安装是否正确。
输入命令: node --version和 npm --version这些命令将分别输出类似于“v0.10.31” 或 “1.4.23”，表示安装已成功。

一旦你安装了这些程序, 接下来你需要安装 grunt和 bower. 你可以通过运行以下命令来进行安装: npm install -g bower grunt-cli
现在你就应该能够执行的命令，“ grunt”和“ bower”。

请注意，我们告诉您的任何命令都必须运行在项目的根目录下。

Bower 依赖管理

使用 bower 来确保你所有的 Javascript 库已下载及更新，在终端或命令提示符下运行命令: bower install。 注意需要运行在你项目的当前工作目录下。

Grunt 编译工具

本项目使用 Grunt 进行编译，使用 Grunt 进行以下任务:

  • Less 编译
  • Bower 依赖管理
  • 发行分支 (要求项目是一个 GIT 仓库)

编译你的项目只需要运行命令 grunt build:angular。 这将编译你的 less 文件， 以及复制你项目中的 bower 依赖。

发行分支允许你修改您的当前项目版本，并使用 git 标记一个新的分支名称。 它还创建了一个基于你之前的提交历史日志。 使用 Grunt 日志模块获取更多的详情，以及得知如何定制。

本地部署

我们很容易在本地服务器上进行部署。 一旦你安装了依赖，并且编译了你的项目， 你可以通过运行命令启动本地服务器 npm start。 这将启动一个本地服务器，并自动打开你的浏览器。

动画

项目中使用的 animate.css是一款超屌、有趣，以及跨浏览器的动画样式引擎。 很好的活跃了主页、滑块以及各个页面元素。

页面切换，请在项目使用 Animate css 中 ui-view 元素， 然后使用 .fade-in, .fade-in-left, .fade-in-right, .fade-in-up, .fade-in-down即可， 如果你想使用大幅度动画，则可以使用 .fade-in-right-big, .fade-in-left-big, .fade-in-up-big, .fade-in-down-big， 也可以使用 .smooth样式类来使动画变得平滑。

核心 Core

你必须先懂得使用 AngularJS， 然后通过 "ui-router" 和 "ui-jq" 来开始你的项目。

界面路由 ui-router

界面路由 ui-router 主要使用了 嵌套路由与 嵌套视图, 请参考 官方文档

界面加载 ui-load

界面加载 ui-load 主要使用了 懒加载来加载 js 和 css 文件。

angular.module('myModule', ['ui.load']);

uiLoad.load( ['path/style.css', 'path/script.js'] ).then(function() {
// 加载成功
}).catch(function() {
// 加载文件时出错
});

界面加载 ui-load 另外还使用了界面路由 ui-router 中的 resolve:{}属性来懒加载 Angular 的指令、控制器以及服务。

jQuery UI 插件 ui-jq

ui-jq调用元素上指定的jQuery函数或插件。 请参考 官方文档 (已失效)

界面模块 ui-module

ui-module为子指令加载其依赖项。 你需要配置 "js/app.js" 文件中的常量 "MODULE_CONFIG"。
比如: 查看 “ui-select2” 指令 “form/element”

界面验证 ui-validate

这个 ui-validate指令使创建自定义验证表达式很容易。 请参考 官方文档

指令 Directives

切换指令 ui-toggle-class

元素被点击时切换样式类，属性 target用于设置目标元素。 <div ui-toggle-class="show" target="#nav"></div>

换挡指令 ui-shift

当进入小屏幕时，仅仅是设置“显示”和“隐藏”可能是远远不够的，或许你还需要换一个父容器，那么试试 ui-shift 吧。
属性 ui-shift 设置以什么方式进入新的父容器，属性 target 设置当小屏幕时要进入哪个父容器。 <div ui-shift="append" target="#nav"></div>

滚动指令 ui-scroll-to

单击时滚动到元素， 属性 target用于设置目标元素。 <a ui-scroll-to="target"></a>

插件 Plugins

所有的 jQuery 插件都依赖于 ui-jq， 你无需为这些插件创建 Angular 指令。 只需要配置 js/app.js 当中的 Angular 常量 JQ_CONFIG， 函数名 => 插件路径即可， 其 js 和 css 文件将自动懒加载。

ngGrid 网格组件

ngGrid 是 AngularJS 为基础的一种网格控件，他的名气较大，大到像 jquery、bootstrap、angularjs 一样，已经大到不适合翻译为中文名称了。

本示例演示了其循环加载的功能。
什么是循环加载： 别看该网格显示了几百几千行数据，但其实只有几十个 <tr> 哦，他会循环的利用这几十个 <tr> 使页面不会过于臃肿、笨拙，这就是循环加载技术 。
那到底有几个 <tr> 呢：他内部有自己的算法，我们就不要去操心这个问题啦！

这个页面并没有什么代码可以学习，要学习该控件如何使用，请查看关键脚本文件 src\controllers\grid.js
或者查看官方文档：http://angular-ui.github.io/ui-grid/
